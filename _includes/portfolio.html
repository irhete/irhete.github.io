Research projects

  Temporal Stability in Predictive Process Monitoring
github | paper

  Alarm-Based Prescriptive Process Monitoring
github | paper
Description: A framework for alarm-based prescriptive process monitoring. The proposed alarming mechanism takes as input a cost model and finds the optimal decision threshold through empirical thresholding.
Skills: cost-sensitive learning, classification, random forest, LightGBM, Python, scikit-learn, pandas, hyperopt
